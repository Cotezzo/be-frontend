Fridge mngmnt (Front-End)

Topics: TypeScript, Node.js, Angular, Front-End

Concept

This project acts as a frontend to the Frige Management (Back-End) project.

My purpose was to create a simple and intuitive interface to the backend that was able to keep track of every operation made by the user, without generating useless traffic and requests to the backend itself. I used Angular to create a solid structure, capable of handling and displaying all the data correctly.

With my very low CSS skills, I managed to create a simple one-page form that implements all the functionalities of the backend (Product deletion, update and insertion).

The final product, once minified by Angular, was directly put in the public folder of the backend, ready to use.
