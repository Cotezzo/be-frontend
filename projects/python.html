Images manipulation

Topics: Python, OpenCV

Concept

This project was born as a joke, since I wanted to mess around with images and animated gifs colours and frames.

The project was abandoned several times in the process, and it's still abandoned to this day, actually.

I managed to create a small console file explorer to choose image files from the program directory, in order to edit them and create new crossfading gifs between images with different styles.

Color manipulation

From the console file manager, the user can manually change the RGB values of the photo, even setting no-sense values that will result in a pretty colorful image.

Crossfading
