Fridge mngmnt (Back-End)

Topics: JavaScript, Node.js, Express.js, Back-End

Concept

This project started because I thought that, at home, we needed a manager to keep track of every product and its expiration date without efforts and prints on paper.
My idea was to make a website on our local network (hosted by me) composed by a database and a backend.

The frontend is a whole different project, analyzed here.

For the sake of learning, I used mongoDB (since I saw it all around but never used it) and the framework Express.js on Node.js to create APIs to "talk" with the frontend and the database.
